This project is a Django-based Shopping Cart application designed to simulate a basic e-commerce system.

It allows users to browse products, add or remove items from their cart, and manage their selections in a simple and intuitive way. The project focuses on implementing core backend logic such as session handling, data management, and dynamic updates.

Key features include:
- Product listing and detail view
- Add to cart and remove from cart functionality
- Cart management with quantity updates
- Basic user interaction flow

This project was built as part of my learning journey in Django, with an emphasis on understanding real-world application structure, debugging, and problem-solving.
